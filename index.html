Intent

A minimalist UI components library with intention.

Github repo

Why it's awesome

Inspired by the design of Stripe.com, Intent is a minimalist UI components library where each shape and colour is intentional. The design is aim to convey information to the user without distraction.

Typography

The base font size is set to 16px, all other elements are relatively sized using the `ex` or `em` units. Noticed that this library does not include styles for h5 and h6 tags, we believe nesting more than 4 levels increase the cognitive load on the user. Intent uses the font family Roboto and includes three font weights, 300, 400 and 500.
