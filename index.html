Intent

A minimalist UI components library with intention

Inspired by the design of Stripe.com, Stripe.com is a minimalistic UI components library where every design detail is intentional. Providing the user a fluid UX while retaining total creative control for the developers.

Typography

The base font size is set to 16px, all other elements are relatively sized using the `ex` or `em` units. Noticed that h5 and h6 are not styled since we believe that you should never use them; they are so similar in size with normal text that it is hard for the user to differentiate them.
